The chat widget is a built-in group chat for everyone sharing a board —
useful for a household or team that already lives on the same dashboard.

## Channels

Chat is organized into channels. Add channels from the widget's settings;
each one gets its own tab inside the widget. By default a channel is open
to everyone who can see the board, but an admin can restrict a channel to
specific members from the channel editor — anyone not on that list won't
see or be able to post in it.

## Sending messages

Type in the composer and send. Messages appear instantly for everyone
viewing the channel. Tabs for channels with unread messages show a small
dot, which clears once you open that tab.

## Markdown

Markdown formatting (bold, italic, strikethrough, inline code) is off by
default and can be turned on per channel from the channel editor — so a
casual channel can stay plain text while another supports formatting.

## Skins

Pick how the chat looks from three built-in skins: iMessage-style bubbles,
WhatsApp-style bubbles, or an IRC/terminal look. The skin is a widget-level
setting.

## Message colors

Each sender gets an automatically assigned color for their name and message
bubble. You can override your own color from your profile — everyone else
sees it applied to your messages immediately. Your own messages always
keep your theme's accent color regardless of your personal color choice.

## Timestamps

Toggle timestamps on messages on or off from the widget's settings.

## History

Messages stay available for as long as the channel's retention setting
allows; an admin sets retention when creating or editing a channel.
