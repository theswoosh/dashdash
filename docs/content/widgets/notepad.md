The notepad widget is a free-text scratchpad you can drop anywhere on a
board — shopping lists, reminders, anything you'd stick on a sticky note.

## Per-board notes

Notepad content belongs to the widget instance, so different notepads on
different boards hold entirely separate text.

## Markdown

Markdown rendering can be turned on for a notepad from its settings, using
the same lightweight formatting (bold, italic, strikethrough, inline code)
as the chat widget. It's off by default, so plain typed text stays plain
unless you opt in.

## Clearing a notepad

Clearing a notepad's content is a destructive action, so it's protected by
hold-to-delete — press and hold the clear control until it completes,
rather than a single click, to avoid wiping notes by accident.
