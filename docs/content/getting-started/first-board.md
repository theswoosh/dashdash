After login you land on the board — a grid of widgets you can rearrange
freely. The essentials:

## The topbar

Across the top you'll find the board name, a clock (if enabled), a search
bar (if search engines are configured), and a config button on the right
that opens the config panel.

## The config panel

The config panel is where everything not on the grid itself lives:

- Your profile — name, email, password, chat color
- Theme picker
- Board settings — name, wallpaper, background
- Admin tools, if you're an admin — users, search engines, config validation
- An info button showing the running version and an update indicator

## Entering edit mode

The board itself is read-only by default so you don't accidentally bump
something while using it. Switch to edit mode from the config panel (or the
edit toggle) to unlock dragging, resizing, and adding widgets.

## Adding a widget

In edit mode, open **Add Widgets** to see the full widget catalog — clock,
healthcheck, stats, bookmarks, search, notepad, iframe, chat, and frame (see
[Widgets]({{< relref "/widgets/" >}}) for what each one does). Drag a widget from the list
onto an open spot on the grid to add it.

Every widget can be configured after it's placed — click its gear icon to
set a title, background color, and widget-specific options like a URL for a
healthcheck tile or timezone for a clock.

## Saving

There's no separate save step. Layout changes, widget additions, and config
edits write straight through as you make them — leave edit mode when you're
happy with the result.
